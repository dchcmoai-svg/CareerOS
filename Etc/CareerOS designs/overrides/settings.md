# Settings Override

**Target Route:** `/settings`
**Primary Doctrine:** Linear + Notion

## Core Focus
Build an **Elite Settings UX**. This is where the user exercises ultimate control.

## UX Rules & Visual Deviations
- **Dense Operational Settings:** Clean sidebar/vertical tabs for Account, Preferences, etc.
- **Privacy UX:** Controls must feel highly tactile and deterministic (e.g. toggling Stealth Mode).
- **Visibility Controls:** Clear visual indicators of who can see what.
- **AI Preference Clarity:** Explicit inline explainers for AI rules so the user never has to guess.
- **Billing Hierarchy:** Premium, transparent presentation of tiers and value.
