# Marketplace Override

**Target Route:** `/marketplace`
**Primary Doctrine:** Linear (with exclusive network energy)

## Core Focus
The Marketplace is an **Exclusive Talent Network Graph**. It is explicitly NOT a social media feed.

## UX Rules & Visual Deviations
- **Anti-Social-Media Doctrine:** No engagement bait, no vanity metrics, no infinite dopamine loops.
- **Discoverability Energy:** Stealth mode toggles and profile push mechanisms must feel highly secure.
- **Recruiter Interactions:** Interactions should feel high-signal, professional, and curated.
- **High-Signal Networking:** Opportunity relevance is prioritized over virality.
- **Passive Confidence UX:** The UI must assure passive candidates that their data is protected and highly valued.
